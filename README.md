# **Spotium**

[![Discord](https://img.shields.io/discord/900260690688897024)](https://discord.gg/TDtncgCPfC)

Spotium is a project coded and developed by vyze and tago. With lots of dedication and almost no updates, Spotium is the easist way to get rid of your spotify ads. We are almost on 24/7 so join the discord for support.  It is so simple to download and setup all for free.  You can check out the website (https://project.spotium.tk) or you can download it here.  **Please be aware that the SpotifySetup is to big of a file to upload to Github.  The Setup file however does contain it, and the Spotify version is extremely important**

## **Requirements**

* **OS**	Windows 10 version 14393.351 or higher
* **Architecture**	x86
